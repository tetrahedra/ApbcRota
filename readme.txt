== Description ==

This widget links to a Google spreadsheet containing the worship rota. It:

- creates a widget to display the next event (Next Sunday at...)
- a link to an existing WordPress page for the full rota


== Installation ==

1. Upload the `ApbcRota` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. On the Widgets page, drag the widget onto the widget area where you want it displayed. You will need to enter at least
the base url for the published Google spreadsheet.
